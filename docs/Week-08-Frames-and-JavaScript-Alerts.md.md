# Week 8 – Frames & JavaScript Alerts

## Objective

Learn how to automate Frames and JavaScript Alerts using Playwright Page Object Model (POM).

## Topics Covered

Implemented:

### Frames

* iframe Handling
* `frameLocator()`
* Frame Navigation
* Frame Heading Validation
* Frame Text Validation

### JavaScript Alerts

* Simple Alert
* Confirm Alert
* Prompt Alert
* `dialog.accept()`
* Confirm Alert Result Validation
* Prompt Alert Result Validation

Automated:

* Navigate to Frames page
* Verify iframe heading visibility
* Verify iframe text
* Handle Simple Alert
* Handle Confirm Alert
* Handle Prompt Alert
* Validate Confirm Alert result
* Validate Prompt Alert result

## Learning Outcome

* Learned iframe handling using `frameLocator()`.
* Automated JavaScript Alerts using Playwright.
* Implemented Confirm and Prompt Alert handling.
* Practiced Page Object Model (POM).
* Improved reusable page methods and locator management.
* Separated application URLs into a configuration file and reusable constants into an application constants file.
