# Browser Windows

## Objective

Learn how to automate browser window interactions using Playwright.

## Topics Covered

Implemented:

* Multiple Tabs
* Browser Context
* Popup Handling
* Child Window Handling
* Window Validation
* `context.waitForEvent('page')`
* `waitForLoadState()`

Automated:

* Open New Tab
* Open New Window
* Switch to Child Window
* Validate Child Window Content
* Close Child Window
* Return to Parent Window

## Learning Outcome

* Learned how to handle multiple browser tabs.
* Understood Browser Context in Playwright.
* Automated popup window handling.
* Validated content in newly opened windows.
* Improved Page Object Model (POM) implementation for browser window scenarios.
* Enhanced reusable methods for window handling.
