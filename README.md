# Playwright Learning POC

## Overview

This repository contains my Playwright learning journey and Proof of Concept (POC) developed using Playwright and TypeScript.

## Technology Stack

- Playwright
- TypeScript
- Node.js

## Project Structure

```
pages/
tests/
utils/
testdata/
```

## Installation

```bash
npm install
```

## Execute Tests

```bash
npx playwright test
```
